FAQs

  • What is this?

    This is a tool that you can use to help researchers study harrassment on Twitter. To let computers learn what consitutes harassment and what doesn't, researchers need lots of examples of harassment, and of normal behaviour. We need your help labeling the examples.

  • How can I help?

    To help out, just rate the tweet below based on how harassing it is. First, read the tweet, then use the buttons below the tweet to submit your evaulation. Thanks to you, we will be better able to spot harassment, and stop it.

  • Where is the data? Can I have the data?

    Once the first thousand tweets have been rated by at least 5 people, I'll add a download link for a daily snapshot of the data. The data will be newline delimited JSON objects with the tweet id, the average harassment rating, and the number of people that have rated the tweet

  • Are PRs welcomed? AKA How can I help?

    Yes. The github is here: https://github.com/hhamilto/twitter_harassment_labeling_service

    Open an issue if you see one, or feel free to just submit a PR. I reserve the right to edit all PRs for clarity and brevity, but in all seriousness, this is a pretty basic project that could be a lot better.

  • What prevents spam/data poisoning?

    Nothing right now. Please see "Are PRs welcomed?".

  • Does this track me?

    Not yet. In the future, I may add cookies so that you aren't presented the same tweet to rate twice.
